An example repo for reproducing Phaser bug

Instructions
---------
1. Execute the `up` script to start HTTP server for serving local files
2. Open http://0.0.0.0:8000 in your browser
3. Wait for player (blue) sprite to land on the ground
4. Press up arrow key and see that the sprite doesn't jump
5. Refresh page and press the up arrow ASAP and see the sprite successfully jump
